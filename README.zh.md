# Weakup

一个高性能、轻量级的 macOS 防休眠实用工具。

## 功能特性

- **一键切换**：单击菜单栏图标即可开启/关闭防休眠
- **菜单栏应用**：驻留在菜单栏，不占用 Dock
- **定时模式**：可设置自动关闭计时器（15分钟、30分钟、1小时、2小时、3小时）
- **视觉状态**：清晰的状态指示（填充/空心的电源圆圈图标）
- **全局快捷键**：`Cmd + Ctrl + 0` 随时切换
- **原生性能**：使用 IOPMAssertion API，零开销
- **SwiftUI + AppKit**：现代简洁的代码库

## 使用方法

1. 点击菜单栏图标切换防休眠状态
2. 点击「设置」访问定时选项
3. 使用 `Cmd + Ctrl + 0` 快捷键随时切换

## 构建和运行

```bash
# 构建 App（包含图标）
./build.sh

# 运行 App
open Weakup.app

# 或者直接双击 Weakup.app
```

构建完成后，你可以将 `Weakup.app` 拖到「应用程序」文件夹，然后通过 Spotlight 或启动台启动。

## 系统要求

- macOS 13.0 或更高版本
- Xcode 命令行工具

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd + Ctrl + 0` | 开启/关闭防休眠 |
